Options

option default description
afterSelect function() {} Function to call after one option is selected.
afterDeselect function() {} Function to call after one option is deselected.

Methods

You must init the multiple select by calling $('#my-select').multiSelect() before you call one of the following methods.


method options example
select value $('#my-select').multiSelect('select', 'fr');
deselect value $('#my-select').multiSelect('deselect', 'fr');
select_all - $('#my-select').multiSelect('select_all');
deselect_all - $('#my-select').multiSelect('deselect_all');
