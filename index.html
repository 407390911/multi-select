Features

  • It works in an unobtrusive fashion.
  • Entirely styleable with CSS.
  • No extra HTML markup needed.
  • You can add as many select as you want on a single page.
  • It provides some callbacks.
  • Methods can be called from outside.

Download and Installation

Simply add "jquery.multi-select.js" to your project.

Download code and examples from my github repository.

Usage

Add a css class to all multiple select you want to customize.
note: an ID is required for your multiple select. 

      
      <select multiple='multiple' class='your_class' id='your_id'>
        ...
      </select>

Add following code to your project

      
      $(function(){
        $('.your_class').multiSelect();
      });
